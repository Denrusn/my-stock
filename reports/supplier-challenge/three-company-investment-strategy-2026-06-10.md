### 🥇 恒立液压（601100）——"核心配置，等待催化剂"

#### 当前定位

| 维度 | 评估 |
|------|------|
| Serenity稀缺层评级 | ★★★★★ 丝杠——物理AI单一最大价值量部件 |
| 当前股价/市值 | ~121元 / ~1400亿 |
| PE(TTM) | ~47x |
| 是否已透支机器人预期 | ⚠️ 部分透支——主业支撑PE~30x，机器人溢价约17x |
| 主业状态 | ✅ 工程机械景气上行（挖掘机销量同比+36%），2026Q1营收32.1亿（+32.5% YoY） |

#### 投资策略建议

**核心判断**: 适合作为物理AI主题的"核心仓位"，逻辑最顺、赔率最大，但要接受验证周期较长（1-2年）。

**策略框架**:
- **仓位角色**: 核心配置（30-40%物理AI敞口）
- **时间维度**: 12-24个月
- **入场逻辑**: 当前机器人溢价不高（丝杠尚未贡献实质营收），主业提供安全垫；下跌有限（年利润~30亿支撑），上涨空间大（丝杠一旦确认放量）
- **止损信号**: 若2026H2季报中丝杠仍无显著收入贡献 + 新剑传动IPO后拿下特斯拉丝杠主要份额 → 需要重新评估

**目标位参考（非投资建议）**:
- 乐观（丝杠确认批量订单）：150-180元
- 中性（丝杠进展顺利但未大规模释放）：120-140元
- 悲观（丝杠进展不及预期+主业下行）：80-100元

**最优操作情景**: 
- 特斯拉Optimus V3 6月底亮相确认丝杠方案（14根+不变或增多）→ 催化剂
- 恒立液压2026H1半年报首次拆分丝杠业务收入（哪怕是千万级）→ 拐点信号
- 实控人汪立平回归后管理层稳定 → 治理风险解除

#### 反方意见

1. **新剑传动IPO**：作为特斯拉丝杠一级供应商启动上市，年产100万套规划，可能分走大部分份额
2. **恒立液压的"核心供应商"定位缺乏官方公告确认**——公司自己的回应仍然是"配合早期研发"
3. **实控人汪立平3月被调查**——虽已恢复正常履职，但治理结构风险不可忽视
4. **丝杠良率爬坡不确定性**——精密丝杠的批量一致性是业内的长期难题
5. **五洲新春/北特科技/贝斯特——多家围剿丝杠赛道**，供给可能提前过剩下压利润率

### 🥈 绿的谐波（688017）——"高弹性、高估值、高波动，更适合交易型"

#### 当前定位

| 维度 | 评估 |
|------|------|
| Serenity稀缺层评级 | ★★★★ 谐波减速器——旋转关节刚需 |
| 当前股价/市值 | ~280元 / ~580亿 |
| PE(TTM) | **~347x**（Yahoo Finance数据） |
| PE(2026E) | ~165x |
| 是否已透支机器人预期 | ✅ **已极度透支**——公司自己提示"非理性炒作" |
| 主业状态 | ⚠️ 体量极小（2026Q1归母净利仅3263万） |

#### 投资策略建议

**核心判断**: 物理AI方向上"确定性最高"（有实际营收确认），但当前估值已把这部分确定性过度定价。适合**等待回调**后再考虑介入，不适合追高。

**策略框架**:
- **仓位角色**: 卫星配置/交易型仓位（不超过15-20%物理AI敞口）
- **时间维度**: 6-12个月（等待估值消化）
- **入场逻辑**: 公司发布"非理性炒作"风险提示——历史经验看，这种公告后1-3个月通常有显著回调。等回调到高盛目标价区间（~134元）或至少PE消化到80-100x再考虑入场
- **不可追高的理由**: 
  - PE(TTM) 347倍——即使2026年利润翻倍到2.5亿，仍对应~200x PE
  - 2026Q1净利润仅3263万——撑不起580亿市值
  - 公司自己已提示风险——说明管理层也认为股价脱离基本面

**目标位参考（非投资建议）**:
- 乐观（人形订单持续爆发）：300-350元
- 中性（业绩稳步释放）：180-220元（接近高盛目标价134-180区间）
- 悲观（Optimus量产延迟+竞争加剧）：100-140元

**最优操作情景**: 
- **先等回调**——6月三连板后的回调通常在2-4周内发生（参考历史规律）
- 等2026半年报确认人形营收从1亿→2亿+，确认高增长持续性
- 如果回调到165-200元区间（PE消化到100-120x），才具备安全边际

#### 反方意见

1. **估值已经进入「如果optimus量产失败就会归零」的定价区间**——这是最大的风险
2. **公司自己提示风险**——三连板后的公告在A股是非常明确的非理性信号
3. **高盛"中性"评级（目标价134.6元）vs 股价~280元**——差距超过100%，说明机构资金与游资之间的巨大分歧
4. **国际竞争格局**：哈默纳科仍是全球老大（70%份额），如果哈默降价竞争，绿的份额和利润率都会受压
5. **技术路线风险**：如果人形机器人更多使用行星减速器/摆线减速器替代谐波，份额被分流

### 🥉 三花智控（002050）——"低弹性防御型、等拐点信号"

#### 当前定位

| 维度 | 评估 |
|------|------|
| Serenity稀缺层评级 | ★★☆ 执行器总成——可替代性较高 |
| 当前股价/市值 | ~44-47元 / ~2000亿 |
| PE(TTM) | **~28x**（三家中最低） |
| PE(2026E) | ~28x |
| 是否已透支机器人预期 | ❌ **未明显透支**——主业年利润40亿已支撑大部分估值 |
| 主业状态 | ✅ 极强——2025年营收310亿，净利润40.63亿（+31% YoY） |

#### 投资策略建议

**核心判断**: 不符合 Serenity "供应链瓶颈猎手"的选股框架（执行器总成可替代性强、证据不足），但作为"稳健的物理AI敞口"对某些投资者有配置价值。如果你想要参与物理AI主题又不想承受绿的谐波的高估值和恒立液压的不确定性，三花可作为"保守仓位"。

**策略框架**:
- **仓位角色**: 防御型配置/指数替代（不超过20%物理AI敞口）
- **时间维度**: 6-12个月（等待拐点）
- **入场逻辑**: 当前股价从高点回撤~26%、PE仅~28倍、年利润40亿提供坚实底部。不利因素（高管减持、大单辟谣）已被部分定价。等以下拐点再入场：
  - 半年报中首次拆分机器人业务收入（不管金额大小，这是信号）
  - 特斯拉Optimus量产启动后三花拿到正式订单公告
- **相比同行的特征**: 上行空间受限于"执行器Tier1"弹性不如零部件；但下行空间也有限（主业支撑）

**目标位参考（非投资建议）**:
- 乐观（确认机器人Tier1订单）：55-65元
- 中性（主业稳步增长+机器人无进展）：45-52元
- 悲观（机器人完全无进展+主业下行）：35-40元

**关键警告**:
- **高盛看空研报**：当前股价隐含Optimus 90-200万台出货预期（30-70%市占率假设），而特斯拉目标到2030年仅100万台
- **高管减持信号**：实控人+5位高管在2026年1月集体减持（"为子女教育"说法市场不买账），这通常发生在内部人认为股价已充分定价后
- **拓普集团TPG的竞争**：拓普也在强力布局执行器总成，两家都在争Tesla执行器份额，最终份额分配悬而未决

#### 反方意见

1. **管理层高位减持是A股最明确的高估信号之一**——实控人+五位董事高管同时减持，不应轻率忽略
2. **大单辟谣的持续性影响**——三花官方公告否认50亿订单，并非所有投资者都知道或充分定价了这个事实
3. **执行器总成竞争格局恶劣**——拓普集团同样强、甚至更激进（50亿投资机器人电驱基地），两家都在争同一个Tesla客户
4. **高盛看空：股价隐含180万台Optimus出货预期**——即使到2030年也未必达到，意味着上行空间有限
5. **机器人业务占收入比可能永远不显著**——三花主业太大（年营收310亿），即使机器人业务做到10亿，占比也仅~3%

## 四、三家综合投资评级

### Serenity 方法论评级体系

> 评分不反映"公司质量好坏"，而反映 **"这家公司在当前价位、当前证据强度下，是否值得分配研究精力和资金去捕捉物理AI主题的弹性"**

| 评级维度 | 权重 | 恒立液压 | 绿的谐波 | 三花智控 |
|---------|:---:|:--------:|:--------:|:--------:|
| 稀缺层约束力 | 30% | ★★★★★ | ★★★★ | ★★ |
| 证据质量 | 25% | ★★★ | ★★★★★ | ★★ |
| 客户紧迫性 | 20% | ★★★★★ | ★★★★ | ★★☆ |
| 当前估值合理度 | 15% | ★★★（部分透支） | ★（极度透支） | ★★★★（合理偏贵） |
| 风险/安全边际 | 10% | ★★（高波动+实控人） | ★（极端估值+高波动） | ★★★★（主业支撑） |
| **综合投资评级** | **100%** | **B+（中性偏正面）** | **C（观望/谨慎）** | **B（中性）** |

### 评级说明

| 评级 | 含义 |
|------|------|
| **A** | 强买入——稀缺层+证据+估值三者共振 |
| **B+** | 中性偏正面——框架上有吸引力，但需等待特定催化剂 |
| **B** | 中性——有配置价值但弹性有限或证据不足 |
| **C** | 观望/谨慎——框架不匹配或估值已极度透支 |
| **D** | 规避——反方证据明显超过正面证据 |

### 具体评级逻辑

#### 恒立液压：B+ 「中性偏正面」

**核心关注**: 丝杠是最硬的约束层，恒立的产能进度领先；当前PE~47x虽然已含部分机器人溢价，但主业提供一定支撑。**等待催化剂确认**（Optimus V3方案、半年报丝杠收入拆分）。

**不建议追高**——但如果出现10-15%的回调（到100-105元区间），是考虑入场的位置。

---

#### 绿的谐波：C 「观望/谨慎」

**核心关注**: 确定性最高（有实际营收），但PE(TTM) 347倍已经为"完美情景"充分定价。公司自己提示风险的情况下，当前追入的风险回报比不理想。

**建议等待回调30-40%以上**（到170-200元区间，PE消化到100-120x）再重新评估。

---

#### 三花智控：B 「中性」

**核心关注**: 稳健但弹性不足，不符合Serenity稀缺框架。更适合作为"保守型物理AI配置"而非"高弹性猎手标的"。高管减持+大单辟谣是明确的不利信号。

**如果持仓成本低或纯属想配置物理AI主题，可以持有**——但不要期待它带来超额收益。

## 七、附注：行业整体风险

> Serenity方法论的核心立场——**物理AI/人形机器人赛道在2026年6月的最大问题不是"好不好"，而是"定价了多少"**

| 风险 | 描述 |
|------|------|
| 量化回调风险 | 机器人指数年内已回撤~20%，第二波系统性的获利了结可能仍未结束 |
| 量产预期过高 | 行业专家判断2026年5万台Optimus"不可能实现，至少还有3年" |
| 缺乏刚性应用场景 | 人形机器人目前ROI不明确，"没有刚性应用场景"是业内真实声音 |
| 估值中枢下移 | 流动性收缩+板块轮动可能导致整个板块的估值中枢系统性下移 |
| 供给过剩风险 | 丝杠/减速器环节已有5-10家企业在扩产，量产后价格竞争可能超预期 |
